Portfolio


LocalScoop

An exercise in API collaboration, LocalScoop answers the age old question: Where can I find some local news, the current weather and the nearest ice cream shop?

WeekNDR

Dedicated to working for the weekend, WeekNDR is a budget tracking application that allows users to set goals and track their progress towards them.

HomePlanit

This application flexes REACT strength in order to create a space for households to stay organized, including persistent lists across pages, a modifiable calendar as well as a quick reference recipe API search.

Friend Finder

FriendFinder is a simple survey app that scores user input against a custom API to find your next best friend.

NYTScraper

An API search app using the New York Times API, this app allows you to search based on keyword or date to find any NYTimes article.

Memory Game

An exercise in REACT building, this is a memory game application that auto-shuffles after each turn.
